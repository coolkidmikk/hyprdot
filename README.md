## Installation

```
git clone --depth 1 https://github.com/coolkidmikk/hyprdot.git ~/hyprdot

cd ~/hyprdot
chmod +x install.sh
./install.sh
```
Keybinding hint: ```Super + /```
Put your wallpaper in ```~/Pictures/Wallpapers/``` and music in ```~/Music```
